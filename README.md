# Industrial Colony 2 Server Pack

Ein umfassendes Minecraft 1.20.1 Forge Modpack für industrielle Automatisierung, epische Luftschiffe, fortschrittliche Waffen und Kolonieaufbau.

## Überblick

**Industrial Colony 2** ist ein Server Pack, das auf Minecraft **1.20.1** mit **Forge** läuft und das Beste aus industrieller Automatisierung, kreativen Baumöglichkeiten, fortgeschrittener Technologie und spannenden Kampfmechaniken vereint. Mit über **175 Mods** bietet dieser Pack eine tiefgreifende Spielerfahrung für technisch versierte Spieler und Abenteurer gleichermaßen.

## Installation

### Server Setup

1. **Voraussetzungen**
   - Java 17 oder höher
   - Mindestens 16GB RAM
   - 10GB freier Festplattenspeicher

2. **Server starten**
   ```bash
   # Windows
   start.bat

   # Linux/Mac
   java -Xmx16G -Xms16G -jar mohist-1.20.1-6e3e2a1.jar nogui
   ```

3. **Konfiguration**
   - Bearbeite `server.properties` für Basis-Einstellungen
   - Passe `bukkit.yml` und `spigot.yml` für Performance an
   - Konfiguriere Mods im `config/` Verzeichnis

## Konfiguration & Anpassungen

### Mod-Konfigurationen
Alle Mod-Konfigurationen befinden sich im `config/` Verzeichnis:
- **Mekanism** - 12 separate Konfigurationsdateien für Maschinen, Generatoren und Tools
- **Applied Energistics 2** - Client und Server Konfigurationen
- **Valkyrien Skies** - Schiffs-Physik und Steuerungseinstellungen
- **Create** - Mechanische Komponenten und Leistungseinstellungen

### Server-Einstellungen
- **server.properties** - Minecraft Vanilla Server-Einstellungen
- **bukkit.yml** - Bukkit Spawn-Limits und Performance
- **spigot.yml** - Entity-Tracking und Weltgenerierung

### KubeJS Scripting
Das KubeJS-System ist vorbereitet für:
- **startup_scripts/** - Spiel-Initialisierungsskripte
- **server_scripts/** - Rezept- und Loot-Modifikationen
- **client_scripts/** - JEI- und Tooltip-Events
